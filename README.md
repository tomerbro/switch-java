# switch-java
### Installation

Make new bin folder under your user:

```sh
$ mkdir bin
$ cd bin
```
In OSX it should be - ```/Users/tomer/bin```

Open (located at root folder):
```
.bash_profile 
```
And add:
```
PATH=$PATH:$HOME/bin
```

Run:
```sh
$ source ~/.bash_profile
$ switch_java.bash {version}
```
